# Exposurebias
# Exposure-bias-with-Counterfactuals
# Anonymous Code Repository

This repository contains code for a research study on offline evaluation under exposure bias in recommender systems.

## Models Implemented

* Neural Collaborative Filtering (NCF)
* Matrix Factorization (MF)
* ExpoMF

## Counterfactual Estimators

* Inverse Propensity Scoring (IPS)
* Self-Normalized IPS (SNIPS)
* Counterfactual Risk Minimization (CRM)
* Doubly Robust (DR)

## Datasets

* Yahoo! R3
* Coat
* KuaiRec

## Usage

Run the main script to reproduce experiments and evaluation results.
